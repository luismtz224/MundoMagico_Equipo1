# [cite_start]Sistema de Gestión para el Salón de Eventos Mundo Mágico [cite: 36]

## Descripción del Proyecto
[cite_start]Este proyecto consiste en el desarrollo de un Sistema de Gestión de Reservaciones y Operación para el salón de eventos infantiles Mundo Mágico[cite: 44]. [cite_start]El propósito principal es digitalizar y automatizar los procesos operativos, como reservas, gestión de personal y control financiero, para reducir ineficiencias y mejorar la satisfacción del cliente[cite: 45, 67].

## Objetivos Específicos
* [cite_start]Implementar un sistema digital de reservaciones con disponibilidad en tiempo real[cite: 69].
* [cite_start]Automatizar el registro y seguimiento de pagos y anticipos[cite: 70].
* [cite_start]Generar cotizaciones y contratos de manera rápida y profesional[cite: 71].
* [cite_start]Implementar recordatorios automáticos de pagos y eventos próximos[cite: 72].

## [cite_start]Arquitectura y Tecnologías 
* [cite_start]**Frontend:** HTML, CSS, JavaScript.
* [cite_start]**Backend:** Python.
* [cite_start]**Base de Datos:** MySQL.
* [cite_start]**Infraestructura:** Servidor en la nube (AWS).

## Metodología
[cite_start]Para el desarrollo de este proyecto se utiliza la metodología RUP (Rational Unified Process)[cite: 96].
