# Sistema de Gestión para el Salón de Eventos Mundo Mágico [cite: 36]

## Descripción del Proyecto
Este proyecto consiste en el desarrollo de un Sistema de Gestión de Reservaciones y Operación para el salón de eventos infantiles Mundo Mágico. 
El propósito principal es digitalizar y automatizar los procesos operativos, como reservas, gestión de personal y control financiero, para reducir ineficiencias y mejorar la satisfacción del cliente.

## Objetivos Específicos
* Implementar un sistema digital de reservaciones con disponibilidad en tiempo real.
* Automatizar el registro y seguimiento de pagos y anticipos.
* Generar cotizaciones y contratos de manera rápida y profesional.
* Implementar recordatorios automáticos de pagos y eventos próximos.

## Arquitectura y Tecnologías 
* **Frontend:** HTML, CSS, JavaScript.
* **Backend:** Python.
* **Base de Datos:** MySQL.
* **Infraestructura:** Servidor en la nube (AWS).

## Metodología
Para el desarrollo de este proyecto se utiliza la metodología RUP (Rational Unified Process).
