# Registro de Uso de Inteligencia Artificial (IA_USO.md)
Este documento detalla cómo se han utilizado herramientas de Inteligencia Artificial durante el desarrollo del proyecto.

## 1. Planificación y Diseño
* **Herramienta:** Gemini / Claude Sonnet
* **Uso:** Apoyo en la estructuración de la documentación técnica y en la definición de la arquitectura de carpetas para el backend y los módulos del sistema. También se utilizó para organizar los requisitos iniciales bajo la metodología RUP.

## 2. Desarrollo (Frontend / Backend)
* **Herramienta:** GitHub Copilot / Claude Sonnet
* **Uso:**
    * **Módulo de Envío de Recordatorios:** Investigación y ejemplos de sintaxis para la integración de servicios de mensajería y correo electrónico en Node.js.
    * **Módulo de Generar Contrato:** Implementación de scripts y reglas CSS específicas para el manejo de la impresión, logrando que el sistema imprima exclusivamente el documento del contrato y no los elementos de la página web.
    * **Base de Datos:** Optimización de las consultas SQL para la gestión de pagos y disponibilidad de eventos en MySQL.

## 3. Pruebas y Documentación
* **Herramienta:** Gemini
* **Uso:** Generación de datos de prueba para validar el comportamiento de los formularios y revisión de la redacción en el manual de usuario para asegurar claridad técnica.
